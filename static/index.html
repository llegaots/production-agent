API keys & configuration

Keys are read from a .env file on the server — not in this browser.

How to connect AI (OpenAI)

  1. In your project folder, copy .env.example to .env
  2. Set OPENAI_API_KEY=sk-…
  3. Optional: OPENAI_MODEL=gpt-4o-mini
  4. Restart the server: python3 -m uvicorn app.main:app --reload

Optional: Supabase database

  1. Add SUPABASE_URL and SUPABASE_SERVICE_ROLE_KEY to .env
  2. Use the service role key from Supabase Dashboard → Settings → API
  3. Never paste service keys into this UI or commit .env to git

Path on your machine: project root / .env
