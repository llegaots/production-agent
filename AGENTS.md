# Writing style

Never use em dashes or en dashes (— or –) anywhere: UI copy, comments, commit messages, agent/system prompts, or seed data. Use a comma, period, colon, parentheses, or a normal hyphen instead. Any AI prompt that produces text shown to users (door notes, lead summaries, grading insights, route summaries, transcripts) must instruct the model the same way.
